Class ALE::UI::Bunch

ALE::UI::Bunch is an adapter to PAC::Bunch container.

Constructor Summary
new ALE::UI::Bunch($size)
Creates the instance of the ALE::UI::Bunch. A variable $size is a total number of particles in the bunch.

void setBeamAttributes($energy,$mass)

Defines mass and energy of the bunch particles.

parameters
 description
$mass particle mass in GeV.
$energy beam energy in GeV.

Example : setBeamAttributes(1.93827231, 0.93827231)

void setPosition($i, $x, $px, $y, $py, $ct, $dp)

Defines 6D coordinates of the i-th particle.

parameters
 description
$i index of particle in this bunch.
$x horizontal position of the particle, referred to the ideal orbit [m].
$px horizontal canonical momentum of the particle referred to the ideal orbit, divided by the reference momentum.
$y vertical position of the particle, referred to the ideal orbit [m].
$py vertical canonical momentum of the particle referred to the ideal orbit, divided by the reference momentum..
$ct negative time difference with respect to the reference particle times the velocity of time [m].
$dp momentum difference, divided by the reference momentum.

Example : setPosition($i, 1.e-2*$i, 0.0, 1.e-2*$i, 0.0, 0.0, 1.e-3*$i)

parameters
 description

 description
