Class ZLIB::Space

Defines the space of Truncated Power Series and initializes a global container of One-Step Index Pointers.

Constructor Summary
Space($dimension, $maxOrder)
Defines the space of Tps objects and initializes the global container of One-Step Index Pointers. The constructor has two arguments: the phase-space dimension ($dimension) and maximum order ($maxOrder) of Tps objects.
