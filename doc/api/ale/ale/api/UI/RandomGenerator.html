Class ALE::UI::RandomGenerator

The RandomGenerator is an adapter to the TEAPOT Gaussian random generator.

Constructor Summary
new ALE::UI::RandomGenerator($iseed)
Creates the instance of the ALE::UI::RandomGenerator. $iseed - an integer seed for initializing the generator.
